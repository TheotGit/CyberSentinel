# CyberSentinel

CyberSentinel es una aplicación progresiva (PWA) orientada a ciberseguridad.  
El proyecto integra un frontend en Angular, un backend en Node.js y una base de datos PostgreSQL, todo orquestado con Docker Compose.

## Arquitectura
- **Frontend:** Angular PWA
- **Backend:** Node.js + Express API
- **Base de datos:** PostgreSQL en contenedor Docker
- **Orquestación:** Docker Compose

## Estado actual
- Base de datos PostgreSQL creada y funcionando en contenedor Docker.
- Próximos pasos: definir esquema de tablas, crear backend en Node.js y frontend en Angular.
